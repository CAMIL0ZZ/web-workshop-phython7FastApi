explicacion de idea para taller

Una aplicación web donde los usuarios registran sus autos modificados (tuning), detallan sus mejoras (mods) y el sistema estima los HP (caballos de fuerza) en base a esas modificaciones.
Además, permite comparar autos entre usuarios.


Permitir a los usuarios:

-Registrar sus autos
-Añadir modificaciones (turbo, ECU, escape, etc.)
-Calcular HP estimados automáticamente
-Comparar autos con otros usuarios
-Ver rankings (opcional)


ENDPOINTS DE USUARIOS

POST /users/ → Crear usuario
GET /users/ → Listar usuarios
GET /users/{user_id} → Obtener usuario específico
DELETE /users/{user_id} → Eliminar usuario


ENDPOINTS DE BUILDS (AUTOS)


POST /builds/ → Crear auto
GET /builds/ → Listar autos
GET /builds/{build_id} → Obtener auto
DELETE /builds/{build_id} → Eliminar auto

ENDPOINTS DE MODIFICACIONES

POST /mods/ → Crear modificación
GET /mods/ → Listar modificaciones
GET /mods/{mod_id} → Obtener modificación
DELETE /mods/{mod_id} → Eliminar modificación
